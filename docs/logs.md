# 升级说明

## 内容

以下列出 dPay `v1` 版本主要的升级内容，供商户参考。

### 🚀 新增功能

- 新增 `订单回调地址` & `提现回调地址` 的全局统一配置能力。
- 新增 `商户密钥` 重置功能。
- 新增 `Google authenticator` 的 `两步安全验证`，有效防范因账密泄漏导致的资产安全问题。
- 新增商户快速创建订单支付功能。
- 新增IP白名单功能，进一步保证接口访问安全。

### 💪 系统优化

- 提升链上交易检测的时效性及稳定性。
- 优化 `官方收银台` 页面交互及性能。
- 优化 `官方文档` 页面的显示及交互。
- 页面不能显示订单总条数bug修复。

> 其它大量贴合商户实际使用需求的升级内容未列出...
